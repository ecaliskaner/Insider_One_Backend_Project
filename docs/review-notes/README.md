# Review Notes

This folder keeps non-runtime case notes out of the repository root.

| File | Purpose |
| --- | --- |
| `prompt.md` | Original implementation prompt notes. |
| `prompt_2.md` | Follow-up implementation prompt notes. |
| `Test.md` | Manual test notes from earlier review passes. |
| `Test_2.md` | Reserved follow-up test note file kept for continuity. |
| `test_cases.md` | Detailed manual and edge-case test plan. |
| `fix.md` | Earlier senior-review findings and fix plan notes. |
